# Angular 2 snippets for emacs #

Just translated
[sublime-angular2-snippets](https://github.com/orizens/sublime-angular2-snippets)
using [sublime2yas](https://github.com/emacsfodder/sublime2yas).


## Installation

```sh
git clone https://github.com/FrancescElies/emacs-angular2-snippets ~/src/emacs-angular2-snippets
```

## Usage

Add `~/src/emacs-angular2-snippets` to `yas-snippet-dirs` via customize.
